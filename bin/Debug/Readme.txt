
	Media Player SMP

1. 概要
  シンプルなメディアプレーヤーです。
  使いやすいようにシンプルにしました。

2. バグ報告などに利用する連絡先
  ブログの任意の記事のコメント欄にお願いします。
  http://kizuki1749.hatenablog.jp/

3. ソフト種別
  このソフトは GPL-3.0 (GNU General Public License v3.0) でフリーソフトとして配布されています。

  本ソフトを使用し、損害が起こっても作者は責任を負いません。
  本ソフトは、自由に複製、改変、再配布が許可されています。(再配布時にGPL3.0ライセンスで配布することが条件です)

4. 動作環境
  このソフトは、Windows Vista 以降のOSで動作します。
  (別途 .NET Framework 4.5.2 のインストールが必要です)

5. インストールとアンインストール
  インストールの際は、zipファイルを任意の場所に展開してください。
  アンインストールの際は、フォルダごと削除で大丈夫です。レジストリなどは使用していません。

6. 使い方
  普通のメディアプレーヤーと変わりありません。

7. 留意事項
  本ソフトは Microsoft より配布されているラッパーDLLを利用しています。
  再配布が許可されているため、3のソフト種別にて説明している再配布についてもDLLを配布することができます。
